Social Behavior

Select a point and use the arrow buttons or keyboard arrow keys to change the date and behavior.
-Up / Down changes disease reproduction by 5%.
-Left / Right changes the date by 5 days.

100% - Normal behavior. The virus will spread quickly.

0% - Total Isolation. The virus cannot spread.

The effect of Social policies and behaviors on the spread of covid19. Combined effect of physical distancing, business closures, and contact tracing, etc...

Testing

Select a point and use the arrow buttons or keyboard arrow keys to change the date and number of tests. Note that the test scale is logarithmic.

The number of tests available per day. Only 15% of tests are positive, and there is a 5 day delay.

When there are not enough tests available, some cases will not be identified and "confirmed" trends may differ from actual infections.

Disease

R - Reporduction Rate - How many new infections are directly caused by an infected person.

R0 - Basic Reporduction Rate - Reproduction rate in a population with no immunity, and before any behavior changes (e.g. social distancing, contact tracing).

First Infection - When was the first infection. This will be earlier than the first confirmed case.

Asymptomatic - Percentage of cases that do not display symptoms

Moderate - Percentage of symptomatic cases that do not require hospitalization.

Severe - Percentage of cases that require hospitalization, but not ventalation. Mortality rate is 10%.

Critical - Percentage of cases that require ventalation. Mortality rate is 50%.
